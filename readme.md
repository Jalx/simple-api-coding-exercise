# Coding Exercise

Coding exercise

## Stack

Expressjs  
Typescript  
Nodemon  
Jest  
Prisma (ORM)  

## Description

Small API that stores People favorite food and movies in a mongodb.  

To run this project:
- cd app
- npm install
- npx prisma generate
- npm run build
- npm run start

To run tests:
- npm run test

You can test the API using the tool [bruno](https://www.usebruno.com). It's a Git integrated API client.
